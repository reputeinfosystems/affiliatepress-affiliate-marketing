== Installation ==
How to install the AffiliatePress Plugin 

Please follow the steps for installation: 

= VIA WordPress =
1. After your download log into your WordPress admin interface
2. After login click on *Plugins* from the left-hand menu
3. Click on *Add new*
4. Click on *Upload*
5. Browse to the directory you downloaded the plugin to and click *Install Now*
6. After WordPress has finished unpacking the file click on *Activate Plugin*
7. After the plugin has been activated you will notice a new menu item on the left-hand navigation labeled AffiliatePress
8. Done. Start exploring the world's best WordPress affiliate plugin!

= VIA FTP =
1. After your download, unzip AffiliatePress from your download .zip
2. Open your FTP client
3. Browse to /your_wp_instalation/wp-content/plugins/ on your hosting server
4. Upload AffiliatePress into this directory
5. Go to your WordPress admin panel
6. Go to plugins and activate AffiliatePress
7. After the plugin has been activated you will notice a new menu item on the left-hand navigation labeled AffiliatePress
8. Done. Start exploring the world's best affiliate plugin!
